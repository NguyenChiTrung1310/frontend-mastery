# Implement bind, call, apply

Re-implement `Function.prototype.bind`, `Function.prototype.call`, and `Function.prototype.apply` from scratch using only the language primitives they rely on.

## Goals
- Understand how `call` sets `this` via property assignment and deletion
- Understand how `apply` spreads an array of arguments into a function call
- Understand how `bind` creates a closure that locks `this`
